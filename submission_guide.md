## 🚀 How to Push All Tasks to GitHub

### Task 1: Complete Git Project Workflow
1. Go to **GitHub.com** and create a **New repository** named `task1_git_workflow`.
2. Open your terminal and run:
   ```powershell
   cd "c:\JobSpot\ost pac assignment\task1_git_workflow"
   git remote add origin https://github.com/YOUR_USERNAME/task1_git_workflow.git

   ```

3. **Screenshot:** Run `git log --oneline --graph --all` in the terminal and take a screenshot.
